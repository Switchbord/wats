WATS is a Bun-first TypeScript toolkit for the WhatsApp Cloud API. It ships as a set of composable packages — a Graph client, a webhook adapter, typed routing, config, CLI, and a standalone service — rather than one monolithic framework, so you adopt only the parts you need.

- [Quickstart](/docs/quickstart) — install the packages and send your first message.
- [Guide](/docs/guide) — the end-to-end walkthrough of clients, webhooks, and handlers.
- [Reference](/docs/reference) — per-package API documentation.
- [Parity](/docs/parity) — what is live-validated, shape-only, or planned against WhatsApp and pywa.
